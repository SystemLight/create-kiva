# create-kiva

kiva模板创建工具

## 使用方法

```
npm install create-kiva -g
create-kiva i <模板名称>
```

## kiva系列模板开发

1. 发布模块包中必须包含template文件夹
2. template文件夹中可以包含gitignore文件，注意不要加 `.` 符号，发布时会被文件系统忽略，模板安装时自动添加
3. 命名规则 `<自定义名称>-kiva`
4. 切记发布之前务必删除所有不需要上传文件，如node_modules文件夹以及IDE创建的临时文件夹
